# Neo Premier League — Brand Kit

**Neo Premier League** is robot football reimagined as a global sport — where engineering,
entertainment, and competition collide.

This page is the **home of the brand**. Everything that defines Neo Premier League — what it stands
for, how it talks, how it looks, and the robot itself — lives here in one organised place. It's
built so **both people and AI tools** can use it. **You don't need to know how to code.**

---

## 👋 New here? Read this first

A few words you'll see, in plain English:

- **This page (a "repository" or "repo")** — just a tidy collection of files, shown on GitHub.
  Click any folder or file to open it. Nothing here will run or break.
- **`.md` files** — plain text files (called *Markdown*). Click one and it shows up as nicely
  formatted text, like a web page. You can read them right here in your browser.
- **A "skill"** — a ready-made page of instructions you give to an AI chatbot (like
  [claude.ai](https://claude.ai) or ChatGPT) so it replies **in the Neo Premier League's voice and
  look**. Think of it as a recipe: paste it in, add a few ingredients, get an on-brand result.
- **The media library** — all the images and video live in a single **Dropbox folder** (link
  below), not on this page, so this stays light and fast.

## 🧭 How to use a skill (step by step)

No coding — if you can copy, paste, and drag a file, you can do this. Example: *"I need a tagline
for the season opener."*

1. **Open the skill.** Go to [`skills/`](skills/), open the folder for the skill you want (e.g.
   `brand-writer`), and click its `SKILL.md` file. **Select all the text and copy it.**
2. **Open an AI chatbot** — [claude.ai](https://claude.ai) or ChatGPT — and **paste** it in.
3. **Attach the brand files it asks for.** Near the top, each skill lists a few files it needs
   (e.g. `voice.md`). Download those from the [`brand/`](brand/) folder and attach them to the chat.
   *(To download a file from GitHub: open it, then click the **download / "Download raw file"**
   button.)*
4. **Tell it what you need** — e.g. *"Write a tagline for the Season 1 opening match."*
5. **Done.** It replies on-brand. Ask for tweaks just like a normal conversation.

> Why attach files? The skills don't copy the brand rules inside themselves — they point to the
> real brand files so there's only ever **one source of truth**. You just bring those files along.

## 🧰 The toolbox (skills)

Small AI helpers, grouped by what they do.

**Make words**
- **brand-writer** — writes on-brand copy for a specific place (tagline, billboard, jersey, social
  post, teaser, announcement, press release).
- **name-forge** — invents on-brand names for teams, robots, tournaments, arenas, and signature moves.
- **voice-rewriter** — takes any text and rewrites it so it sounds like Neo Premier League.
- **campaign-strategist** — comes up with marketing campaign ideas for a goal you give it.

**Make pictures** *(these write the "prompt" you paste into an image generator like Midjourney)*
- **image-art-director** — creates the prompt for an on-brand image of the robot.
- **key-art-composer** — plans a finished poster/graphic: where the headline and graphics sit on
  top of an image.

**Check it's on-brand**
- **copy-reviewer** — scores a piece of writing and says on-brand / borderline / off-brand, with fixes.
- **image-reviewer** — looks at an image and judges whether it's on-brand.
- **robot-fidelity-checker** — checks the robot in an image matches the official robot.

**Evolve the brand**
- **brand-evolver** — helps update the brand over time and keeps a dated record of what changed.

## ✨ The one idea behind all of this

The brand is written down **once**, in plain files, so everyone — and every AI tool — works from
the same truth. Generate something, check it against the brand, ship it. No scattered docs, no
guesswork, no "which version is right?"
