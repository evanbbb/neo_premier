# assets/ — curated hero assets

The single place to grab **approved** Neo Premier League visuals. This is a small, deliberate set
— not a dumping ground. Exploration files, duplicates, and large archives stay out.

Subfolders:

- `robot/` — the mascot: hero shots, turnaround, uniform.
- `style/` — the look: textures, type treatments, shape language.
- `reference/` — references that inform the brand (kept for context).
